¿Como usar git?


Solo 1 vez por proyecto

  1. git init
  2. git status -s
  3. git add .
  4. git commit -m "inicio proyecto nuevo"
  5. https://github.com/
  6. git remote add origin http:git......
  7. git push -u origin master

¿Para que sirven?

  1. iniciar
  2. consulta si hay archivos modificados
  3. salva todo el proyecto completito
  4. despues de cualquier cambio SIEMPREagregar comentario
  5. logear y crear nuevo respositorio
  6. linkea con la direccion de respositorio
  7. salva archivos en la nube

Uso regular

  1. git status -s
  2. git add .
  3. git add index.html
  4. git commit -m "comentario 1"
  5. git log --oneline
  6. git reset --hard a532423

  7. git push

lo que hacen

  1. consulta si hay archivos modificados
  2. salva todo el proyecto completito
  3. salva uno en especifico index.html en este caso
  4. despues de cualquier cambio SIEMPRE agregar comentario
  5. consulta cuales versiones hay subidas
  6. regresa a la version a a532423 en este caso

  7. sube todo a la nube
